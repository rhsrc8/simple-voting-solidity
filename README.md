# Simple Voting Smart Contract

A basic Ethereum smart contract written in Solidity.

## Features
- One vote per address
- Candidate validation
- Public vote counts

## Tech Stack
- Solidity ^0.8.20
- Remix IDE

## How to Use
Deploy the contract with a list of candidate names.
